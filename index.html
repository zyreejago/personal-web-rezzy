About Me!

I'am Mohammad Rezzy Al Zamzammi

Nama saya Mohammad Rezzy Al Zamzammi, namun saya lebih sering dipanggil Rezzy. Saya berasal dari Jembrana, Negara, Bali, dan saat ini saya sedang menempuh studi di Universitas Pendidikan Ganesha (Undiksha), jurusan Teknik Informatika, semester 5.

Sejak awal saya masuk ke Undiksha, saya sudah memiliki minat yang besar dalam bidang coding, terutama dalam pemrograman web. Minat ini semakin berkembang ketika saya mendapatkan mata kuliah Pemrograman Web pada semester 3, yang berhasil saya selesaikan dengan hasil yang sangat memuaskan. Pada semester 4, saya melanjutkan dengan mata kuliah Pemrograman Mobile, dan kembali mendapatkan nilai yang sangat baik. Di semester 5 ini, saya memilih konsentrasi Rekayasa Perangkat Lunak (RPL) untuk lebih memperdalam pengetahuan saya.

Dalam perjalanan saya di dunia pemrograman, saya telah mempelajari berbagai bahasa dan teknologi seperti HTML, CSS, JavaScript, PHP, Python, dan C++. Selain itu, saya juga sudah mengeksplorasi framework Laravel. Saat ini, saya sedang fokus mendalami library React.js karena, setelah mempelajari JavaScript, saya menemukan bahwa bahasa ini cukup mudah dipahami dan sangat fleksibel untuk pengembangan front-end. Untuk bagian back-end, saya sudah familiar dengan penggunaan MySQL dan Firebase. Saya terus mengembangkan keterampilan dan pengetahuan saya di dunia pemrograman, dengan harapan dapat menghadapi tantangan baru yang muncul dalam industri teknologi.
